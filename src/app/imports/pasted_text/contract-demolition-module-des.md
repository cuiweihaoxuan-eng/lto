一、3.3.8.1 合同解构模块 页面设计
页面 1：合同解构列表页
页面定位
合同解构模块的统一入口，集中展示所有项目合同的解构状态、审批进度、变更记录，支持全量合同的查询、筛选与批量操作，适配项目经理、部门负责人、法务、财务等多角色查看需求。
页面布局与包含内容
顶部：查询筛选区 + 高频操作按钮区
中间：合同解构数据列表区，支持行内快捷操作
底部：分页控件 + 数据统计栏（合同总数、待解构、审批中、已生效、变更中数量）
核心字段明细
表格
字段名称	字段属性	字段说明
序号	🔒只读	系统自动生成
项目编号	🔒只读	对接采购系统同步，项目唯一标识
项目名称	🔒只读	中标项目全称
合同编号	🔒只读	正式合同唯一编号
合同名称	🔒只读	合同全称
合同总金额（元）	🔒只读	合同含税总金额
项目工期	🔒只读	合同约定的开工 - 竣工日期
解构状态	🔒只读	枚举值：待解构、审批中、已生效、已驳回、变更中、已作废
审批进度	🔒只读	实时展示当前审批节点 / 总节点数
经办人	🔒只读	解构发起人的姓名 + 所属部门
最近更新时间	🔒只读	最后一次编辑 / 审批的时间
操作列	-	行内快捷操作按钮
操作按钮及能力说明
表格
按钮名称	可见权限	按钮能力说明
新增解构	项目经理 / 经办人	点击跳转至【合同解构编辑页】，自动带入当前登录人信息，支持选择已中标、未发起解构的正式合同，发起全新解构流程
批量导出	所有查看角色	支持勾选多条数据，一键导出选中合同的解构基础信息 Excel 报表，导出记录自动留存
刷新	所有查看角色	一键刷新列表数据，同步最新解构状态与审批进度
查看	所有查看角色	点击跳转至【合同解构详情页】，只读模式查看完整解构内容、审批记录、操作日志
编辑	项目经理 / 经办人	仅「待解构、已驳回」状态的合同可见，点击跳转至【合同解构编辑页】，可修改解构内容后重新提交
变更申请	项目经理 / 经办人	仅「已生效」状态的合同可见，点击跳转至【合同变更申请页】，发起合同解构内容变更流程
作废	部门负责人 + 管理员	仅「待解构、已驳回」状态可见，点击弹出确认弹窗，填写作废原因后提交，作废后合同不可再编辑，操作记录全量留痕
搜索 / 筛选	所有查看角色	支持按项目名称 / 编号、合同编号、解构状态、经办人、签约日期范围进行精准 / 模糊查询，支持筛选条件重置
页面 2：合同解构编辑页（核心操作页）
页面定位
合同条款拆解、业务数据关联的核心操作页面，分为 2 个固定 Tab 页，完全匹配需求中的两大核心功能，支持解构内容的录入、编辑、关联配置与提交审批。
页面布局与包含内容
顶部：页面标题 + 合同基础信息栏（只读，锁定合同核心信息）+ 提交 / 保存按钮
中间：Tab 切换区，固定 2 个 Tab：「Tab1 合同条款拆解」、「Tab2 业务数据关联」
底部：操作日志栏（实时展示编辑记录）
核心字段明细
【Tab1 合同条款拆解】字段
表格
字段名称	字段属性	字段说明
一、合同基础资料区	-	-
合同编号	🔒只读	从采购系统同步，不可修改
项目名称	🔒只读	锁定中标项目名称
签约甲乙双方	🔒只读	合同甲方、乙方全称
合同总金额（元）	🔒只读	含税总金额，锁定
合同工期	🔒只读	合同约定开工、竣工日期
合同资料附件	✏️可编辑	支持上传合同原件、中标通知书、招投标文件、补充协议，支持在线预览、删除、新增，* 必填至少上传正式合同原件
二、合同核心条款拆解区	-	-
里程碑节点编号	🔒只读	系统自动生成，如 LCB-001
* 里程碑节点名称	✏️可编辑	如：初验完成、终验完成、质保期满

* 节点付款比例	✏️可编辑	合同约定的该节点付款占比，自动计算对应金额，所有节点比例总和需 = 100%，系统自动校验
节点付款金额（元）	🔒只读	系统根据总金额 × 付款比例自动计算
* 节点验收标准	✏️可编辑	合同约定的该节点验收合格标准，支持富文本编辑
对应违约责任条款	✏️可编辑	录入合同中该节点对应的违约罚则、履约要求
质保期约定	✏️可编辑	合同约定的质保期时长、起算时间、质保要求
廉政 / 保密 / 合规条款	✏️可编辑	录入合同约定的廉政、保密、国资监管相关条款
【Tab2 业务数据关联】字段
表格
字段名称	字段属性	字段说明
一、交付物与项目任务绑定区	-	-
任务编号	🔒只读	系统自动生成 WBS 任务编号
* 任务名称	✏️可编辑	对应里程碑节点的拆解任务名称
对应里程碑节点	✏️可编辑	下拉选择 Tab1 中拆解的里程碑节点，实现绑定
* 必填交付物清单	✏️可编辑	支持新增 / 删除行，录入交付物名称、格式要求、提交时限，与当前任务强绑定
二、收款节点与计收规则匹配区	-	-
对应付款节点	✏️可编辑	下拉选择 Tab1 中拆解的付款里程碑节点，一一绑定
* 收入计收规则	✏️可编辑	下拉选择：里程碑验收完成一次性计收、完工百分比法计收，匹配政企会计准则
* 计税规则	✏️可编辑	下拉选择：一般计税（6%/9%/13%）、简易计税，自动匹配税率
三、验收标准与核验指标库关联区	-	-
对应验收节点	✏️可编辑	下拉选择 Tab1 中拆解的验收里程碑节点
验收项清单	✏️可编辑	支持从系统核验指标库一键导入合同约定的验收标准，支持新增 / 编辑 / 删除验收项，每个验收项明确：指标名称、合格阈值、核验方式
四、违约责任与预警规则关联区	-	-
违约触发场景	✏️可编辑	下拉选择：工期延误、质量不达标、交付物缺失
对应合同违约条款	🔒只读	自动同步 Tab1 中录入的违约责任条款
* 预警风险等级	✏️可编辑	下拉选择：一般、较大、重大、特别重大，自动绑定对应预警规则
* 通知责任人	✏️可编辑	多选配置该违约场景触发后，需通知的岗位 / 人员
操作按钮及能力说明
表格
按钮名称	可见权限	按钮能力说明
保存草稿	项目经理 / 经办人	点击后实时保存当前编辑的所有内容，不触发审批流程，可后续继续编辑，保存后状态为「待解构」
智能提取	项目经理 / 经办人	仅上传合同附件后可见，点击后调用 OCR 引擎，自动识别合同附件中的核心条款，填充至对应拆解字段，减少人工录入，提取后支持人工修正
导入模板	项目经理 / 经办人	点击后支持下载标准化合同解构模板，填写完成后一键导入，自动填充所有拆解字段，支持导入后编辑
新增行	项目经理 / 经办人	针对里程碑、任务、交付物、验收项等清单，点击后新增空白行，支持批量新增
删除行	项目经理 / 经办人	针对清单行，勾选后点击删除，弹出确认弹窗，确认后删除对应行，系统自动校验必填行不可删除
提交审批	项目经理 / 经办人	点击后系统自动校验所有必填字段是否完整、付款比例总和是否合规，校验通过后，按预设流程提交至部门负责人→法务→财务→分管领导审批，提交后页面转为只读模式，状态更新为「审批中」
重置	项目经理 / 经办人	点击后清空当前所有编辑内容，恢复至初始状态，弹出二次确认弹窗，避免误操作
附件预览	所有查看角色	点击已上传的附件，在线打开预览，支持下载
页面 3：合同解构详情页（只读模式）
页面定位
用于全角色查看完整的合同解构内容、审批记录、变更历史、操作日志，无编辑权限，满足审批、审计、监督查看需求。
页面布局与包含内容
顶部：合同基础信息栏 + 解构状态标签
中间：4 个固定 Tab：「合同条款拆解详情」、「业务数据关联详情」、「审批记录」、「变更历史」
底部：操作日志全记录区
核心字段明细
核心字段与编辑页完全一致，全部为只读锁定状态，不可编辑
新增审批记录字段：审批节点、审批人、审批意见、审批时间、审批状态（通过 / 驳回 / 待审批）
新增变更历史字段：变更申请编号、变更内容、变更原因、审批状态、生效时间
操作按钮及能力说明
表格
按钮名称	可见权限	按钮能力说明
导出详情	所有查看角色	一键导出完整的合同解构详情 PDF 文件，自动加盖电子水印，满足归档、审计需求
打印	所有查看角色	调用打印控件，支持详情页全内容打印
变更申请	项目经理 / 经办人	仅「已生效」状态可见，点击跳转至【合同变更申请页】
关闭	所有查看角色	点击关闭详情页，返回列表页

页面定位

页面布局与包含内容
顶部：原合同基础信息只读栏 + 变更申请基础信息区
中间：变更内容编辑区，与原解构内容自动对比，标红变更部分
底部：变更原因 + 附件上传区 + 提交按钮

* 变更申请编号	🔒只读	系统自动生成，唯一标识

二、3.3.8.2 实施进度监控模块 页面设计

页面布局与包含内容
顶部：筛选区（按时间周期、部门、项目类型、项目状态筛选）+ 刷新按钮
中间：核心指标卡片区 + 可视化图表区，分为上下两排
上排：核心指标卡片，共 6 张
下排：4 个可视化图表模块
