# ICT项目预算明细 PRD

## 需求背景
展示ICT项目预算明细数据，帮助管理者了解项目预算执行情况。

## 前端页面描述
- 组件：IctBudgetDetail
- 位置：作为页面内容显示

## 功能描述

### 页面布局
| 区域 | 组件 | 说明 |
|------|------|------|
| 查询区 | 表单 | 多维度筛选 |
| 操作区 | 按钮组 | 导出、刷新 |
| 数据表格 | 17列表格 | 5个分组 |

### 查询字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 项目名称 | Input | 否 | 空 | - |
| 省份 | Select | 否 | 全部 | - |
| 项目类型 | Select | 否 | 全部 | - |
| 预算状态 | Select | 否 | 全部 | 预算内/超预算/待确认 |

### 表格列（17列，5个分组）
| 分组 | 列数 | 说明 |
|------|------|------|
| 项目信息 | 4列 | 项目编号、名称、省份、客户 |
| 金额信息 | 4列 | 总投资、总预算、已执行、剩余 |
| 预算信息 | 4列 | 预算科目、明细金额、占比 |
| 执行情况 | 3列 | 执行进度、执行偏差、预警级别 |
| 状态 | 2列 | 预算状态、操作 |

### 预算状态Badge
| 状态值 | 颜色 | 说明 |
|--------|------|------|
| 预算内 | 绿色 | 执行金额在预算范围内 |
| 超预算 | 红色 | 执行金额超出预算 |
| 待确认 | 灰色 | 待确认状态 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 查询 | 操作区 | Primary | 执行筛选查询 |
| 重置 | 操作区 | Outline | 重置筛选条件 |
| 导出数据 | 操作区 | Outline | 导出预算明细 |
| 刷新 | 操作区 | Outline | 刷新列表 |
