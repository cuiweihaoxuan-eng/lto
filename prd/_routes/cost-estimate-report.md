# 成本暂估列账率报表 PRD

## 需求背景
分析成本暂估列账率，包含产数服务和设备销售两大业务板块，帮助财务管理者了解成本确认进度。

## 前端页面描述
- 组件：CostEstimateReport
- 位置：作为页面内容显示

## 功能描述

### 页面布局
| 区域 | 组件 | 说明 |
|------|------|------|
| 查询区 | ReportTemplate | 通用模板筛选 |
| 操作区 | 按钮组 | 导出、刷新 |
| 数据表格 | 23列表格 | 3个分组 |

### 查询字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 项目名称 | Input | 否 | 空 | - |
| 省份 | Select | 否 | 全部 | - |
| 业务板块 | Select | 否 | 全部 | 产数服务/设备销售 |
| 时间范围 | DateRangePicker | 否 | 当前年度 | - |

### 表格列（23列，3个分组）
| 分组 | 列数 | 说明 |
|------|------|------|
| 项目基本信息 | 9列 | 项目编号、名称、省份、客户等 |
| 产数服务 | 7列 | 预算成本、暂估成本、列账率等 |
| 设备销售 | 7列 | 预算成本、暂估成本、列账率等 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 查询 | 操作区 | Primary | 执行筛选查询 |
| 重置 | 操作区 | Outline | 重置筛选条件 |
| 导出数据 | 操作区 | Outline | 导出报表数据 |
| 刷新 | 操作区 | Outline | 刷新列表 |
