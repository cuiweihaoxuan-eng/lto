# 建设未转固无费用报表 PRD

## 需求背景
监控建设类项目中未转固且无费用发生的情况，帮助管理者发现项目执行异常。

## 前端页面描述
- 组件：ConstructNotFixedNoExpense
- 位置：作为页面内容显示

## 功能描述

### 页面布局
| 区域 | 组件 | 说明 |
|------|------|------|
| 查询区 | ReportTemplate | 按维度筛选 |
| 操作区 | 按钮组 | 导出、刷新 |
| 数据表格 | ReportTemplate | 23列表格，5个分组 |

### 查询字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 账期 | Select | 否 | 空 | - |
| ict项目类型 | Select | 否 | 全部 | - |
| 公司类型 | Select | 否 | 全部 | - |

### 表格列（23列，5个分组）
| 分组 | 列数 | 说明 |
|------|------|------|
| 账期信息 | 1列 | 账期 |
| 基础数据 | 4列 | ICT项目来源、工程项目来源、是否有数据、公司类型 |
| ICT项目信息 | 8列 | ICT项目编码、名称、预算、账套、类型、累计收入、首次收入时间、累计成本 |
| 工程项目信息 | 10列 | 工程项目编码、名称、立项时间、账套、预转固日期、转固日期、状态、批复金额、投资占比、累计投资 |
| 稽核结果 | 1列 | 稽核结果 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 查询 | 操作区 | Primary | 执行筛选查询 |
| 重置 | 操作区 | Outline | 重置筛选条件 |
| 导出数据 | 操作区 | Outline | 导出报表数据 |
