# 业财全流程宽表 PRD

## 需求背景
展示项目全生命周期业财数据，包含收入、收款、支出、付款、投融资五大业务板块，实现业财一体化可视化。

## 前端页面描述
- 组件：FullFlowTable
- 位置：作为页面内容显示

## 功能描述

### 页面布局
| 区域 | 组件 | 说明 |
|------|------|------|
| 查询区 | 表单 | 多条件筛选 |
| 操作区 | 按钮组 | 导出、自定义列 |
| 数据表格 | 102列超宽表 | 三级表头合并 |
| 详情侧边栏 | 侧边栏弹窗 | 查看单项目完整信息 |

### 查询字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 项目名称 | Input | 否 | 空 | - |
| 省份 | Select | 否 | 全部 | - |
| 项目类型 | Select | 否 | 全部 | - |
| 时间范围 | DateRangePicker | 否 | 空 | - |

### 表格列（102列，七大分组）
| 分组 | 列数 | 说明 |
|------|------|------|
| 项目基本信息 | 20列 | 项目编号、名称、省份、客户等 |
| 收入 | 20列 | 概算/预算/结算/决算收入金额 |
| 收款 | 20列 | 概算/预算/结算/决算收款金额 |
| 支出 | 20列 | 概算/预算/结算/决算支出金额 |
| 投资 | 10列 | 投资相关金额 |
| 付款 | 12列 | 概算/预算/结算/决算付款金额 |

### 三级表头结构
- 第一级：六大分组（项目基本信息/收入/收款/支出/投资/付款）
- 第二级：各分组内按概算/预算/结算/决算分类
- 第三级：具体金额字段

### 每项目数据行（4行）
| 行类型 | 说明 |
|--------|------|
| 概算行 | 项目概算数据 |
| 预算行 | 项目预算数据 |
| 结算行 | 项目结算数据 |
| 决算行 | 项目决算数据 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 查询 | 操作区 | Primary | 执行筛选查询 |
| 重置 | 操作区 | Outline | 重置筛选条件 |
| 导出 | 操作区 | Outline | 导出全部数据 |
| 自定义列 | 操作区 | Outline | 打开自定义列弹窗 |
| 列宽重置 | 操作区 | text | 恢复默认列宽 |
| 查看详情 | 表格行 | text | 打开详情侧边栏 |

### 交互功能
| 功能 | 说明 |
|------|------|
| 列宽拖动 | 支持拖动调整列宽 |
| 自定义列 | 选择显示哪些列 |
| 详情侧边栏 | 点击行展示单项目完整信息 |
| 行展开 | 每项目4行数据展示 |

## 数据流图
```
后端数据接口 → 102列宽表渲染 → 用户自定义列配置 → 详情侧边栏
```
