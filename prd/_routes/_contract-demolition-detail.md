# 合同解构详情 PRD

## 需求背景

### 痛点
- **问题现象**：需要查看合同解构的完整信息，包括基本信息、解构明细、审批流程，当前无统一查看入口
- **发生频率**：高（每日多次）
- **当前 workaround**：通过多个页面或导出Excel查看

### 业务目标
- **量化指标**：合同解构信息查看效率提升，减少页面跳转次数
- **目标期限**：2026-Q2

### 涉及系统/模块
- **模块名称**：合同解构详情
- **变更类型**：新增
- **对接接口**：合同解构详情查询接口

## 用户故事

### 故事1
- **角色**：项目经理/财务人员/审批人
- **功能**：在详情页面查看合同解构的完整信息，包括基本信息、解构明细、审批流程，可进行编辑/申请变更操作
- **收益**：信息集中查看，操作一站完成
- **验收条件**：状态为待解构或已驳回时显示编辑解构按钮；状态为已生效时显示申请变更按钮

## 数据流图

### 接口1：获取合同解构详情
- **请求路径**：`GET /api/contract-demolition/detail`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `demolitionId` - 类型：字符串；必填：是；来源：路由参数；校验：
- **响应字段**：
  - `record` - 类型：对象；描述：合同解构记录
  - `record.projectCode` - 类型：字符串；描述：项目编号
  - `record.contractAmount` - 类型：数字；描述：合同总金额
  - `record.status` - 类型：字符串；描述：解构状态
  - `record.demolitionItems[]` - 类型：数组；描述：解构明细列表
  - `record.approvalHistory[]` - 类型：数组；描述：审批流程记录
- **存储位置**：数据库表 contract_demolition / contract_demolition_item / approval_history
- **错误码**：
  - `401` - `无权限`
  - `404` - `解构记录不存在`
  - `500` - `服务器异常`

### 数据刷新点
- **刷新时机**：页面加载时
- **影响字段**：基本信息/解构明细/审批流程

## 验收标准

### 正常流程
- [ ] **操作**：从合同解构列表点击"查看详情" → **预期**：页面进入详情模式，显示完整基本信息和解构明细
- [ ] **操作**：查看状态Badge → **预期**：不同状态显示对应颜色（待解构=灰/审批中=蓝/已生效=绿/已驳回=红）
- [ ] **操作**：查看解构明细 → **预期**：表格显示所有解构科目，底部有合计行
- [ ] **操作**：查看审批流程 → **预期**：时间线展示各审批步骤及状态
- [ ] **操作**：点击返回列表 → **预期**：返回合同解构列表页

### 异常流程
- [ ] **操作**：解构编号不存在 → **预期**：显示404页面或返回错误
- [ ] **操作**：状态=待解构时查看 → **预期**：显示编辑解构按钮
- [ ] **操作**：状态=已生效时查看 → **预期**：显示申请变更按钮
