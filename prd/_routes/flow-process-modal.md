# 流转流程模态框 PRD

## 需求背景
展示商情或项目的处理流程，以时间轴形式展示历史操作记录。

## 前端页面描述
- 组件：FlowProcessModal
- 位置：作为弹窗显示

## 功能描述

### 内容结构
| 区块 | 说明 |
|------|------|
| 流程时间轴 | 垂直时间轴展示流程记录 |
| 操作卡片 | 每条记录包含时间、操作人、操作内容 |

### 流程记录字段
| 字段名 | 说明 |
|--------|------|
| 时间 | 操作时间 |
| 操作人 | 执行操作的人员 |
| 执行操作 | 操作类型名称 |
| 操作内容 | 详细描述 |
| 接收人 | 接收通知的人员（可选） |

### 操作按钮
| 按钮名称 | 样式 | 说明 |
|----------|------|------|
| 确定 | Primary | 关闭弹窗 |
