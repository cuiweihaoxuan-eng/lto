# ICT项目分成异常报表 PRD

## 需求背景

### 痛点
- **问题现象**：ICT 分成项目中，存在 DICT 收入小于 DICT 减收、已计划减收但实际未减收等异常情况，缺乏统一的报表来发现和跟踪这些问题项目的处理进度。
- **发生频率**：高
- **当前 workaround**：财务人员定期通过 Excel 比对 DICT 收入与 DICT 减收数据，识别异常后通过线下方式通知项目经理跟进，过程不可追溯。

### 业务目标
- **量化指标**：异常项目识别覆盖率 100%；稽核结果自动生成准确率 > 95%
- **目标期限**：2026-Q2 上线

### 涉及系统/模块
- **模块名称**：ICT项目分成异常报表（IctShareAbnormalReport）
- **变更类型**：修改
- **对接接口**：`GET /api/reports/ict-share-abnormal`

## 用户故事

### 故事1
- **角色**：财务分析人员
- **功能**：通过月报/年度 Tab 切换，查看 ICT 分成项目的异常数据，月报展示当月 DICT 收入/减收/计划金额和进度，年度展示累计/本年数据
- **收益**：一个报表覆盖月度异常和年度异常，月报/年报数据结构差异清晰，减少切换成本
- **验收条件**：默认 Tab 为月报，选择年度 Tab 后表格列从 28 列变为 33 列，稽核结果列从 1 列变为 2 列

### 故事2
- **角色**：财务人员
- **功能**：系统根据预置规则自动计算稽核结果，在稽核结果列中显示异常描述（如"当月DICT收入金额小于DICT减收金额"），异常项目一目了然
- **收益**：自动发现异常，无需人工逐行比对，快速定位问题项目
- **验收条件**：当 DICT 收入 < DICT 减收时，该行稽核结果列显示"当月DICT收入金额小于DICT减收金额"红色提示

### 故事3
- **角色**：项目经理
- **功能**：按项目维度查看分成合同信息（前向合同 + 分成合同）和收入/减收进度，支持按进度百分比范围筛选
- **收益**：在分成合同视角下查看项目，快速了解分成合作条款及执行进度
- **验收条件**：筛选"本月收入进度 < 5%"后，仅显示进度低于 5% 的异常项目，稽核结果高亮

### 故事4
- **角色**：财务人员
- **功能**：点击"导出"按钮，将当前筛选条件下的报表数据导出为 Excel，供管理层汇报使用
- **收益**：一键导出标准格式报表，减少手工整理报表的时间
- **验收条件**：点击"导出"按钮后，浏览器下载 Excel 文件，内容包含所有可见列数据

## 验收标准

### 正常流程
- [ ] **操作**：进入页面，默认 Tab 为"月报" → **预期**：页面标题显示"ICT项目分成异常报表-月报"，表格显示月报数据，共 28 列
- [ ] **操作**：点击"年度" Tab → **预期**：页面标题显示"ICT项目分成异常报表-年度"，表格切换为年报数据，列变为 33 列，稽核结果列变为 2 列
- [ ] **操作**：页面加载 → **预期**：默认显示基础条件+项目信息，不显示前向合同信息、分开合同信息、稽核
- [ ] **操作**：点击"展开更多条件" → **预期**：显示前向合同信息、分开合同信息、稽核条件
- [ ] **操作**：点击"收起更多条件" → **预期**：隐藏前向合同信息、分开合同信息、稽核条件，恢复默认折叠状态
- [ ] **操作**：月报基础条件包含"账期"下拉框 → **预期**：下拉选项包含 2026-01、2026-02 等账期
- [ ] **操作**：年报前向合同信息包含"前向合同签约日期"日期范围控件 → **预期**：可选择开始和结束日期
- [ ] **操作**：年报分成合同信息包含"累计DICT减收金额"数值范围输入框 → **预期**：可输入最小值和最大值
- [ ] **操作**：年报稽核包含"累计数据稽核结果"和"本年数据稽核结果"两个下拉框 → **预期**：两个下拉框独立选择
- [ ] **操作**：填写"本月收入进度=7.8"后点击查询 → **预期**：表格仅显示本月收入进度为 7.8% 的项目
- [ ] **操作**：点击"查看"按钮 → **预期**：显示详情侧边栏（左侧40%宽度）
- [ ] **操作**：鼠标悬浮到表格行 → **预期**：不自动弹出详情浮层
- [ ] **操作**：点击"导出"按钮 → **预期**：浏览器下载 Excel 文件
- [ ] **操作**：某行 DICT 收入 < DICT 减收 → **预期**：稽核结果列显示红色告警文本"当月DICT收入金额小于DICT减收金额"

### 异常流程
- [ ] **操作**：查询接口返回 401 → **预期**：页面顶部显示"未授权，请重新登录"
- [ ] **操作**：查询接口返回 500 → **预期**：表格区域显示"服务器异常，请稍后重试"
- [ ] **操作**：网络断开时点击查询 → **预期**：显示"网络异常"提示
- [ ] **操作**：输入的进度百分比为负数 → **预期**：字段下方显示"请输入正数"
