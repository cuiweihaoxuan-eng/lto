# 业财全流程宽表 PRD

## 需求背景

### 痛点
- **问题现象**：项目经理、财务人员需要在一个界面综合查看项目的收入、收款、支出、付款全流程财务数据，当前需要分别在多个系统中查看后再手工汇总，跨系统数据关联困难，数据时效性差。
- **发生频率**：高
- **当前 workaround**：管理员从多个系统分别导出 Excel，手工整理后生成综合报表，耗费大量人力且数据可能不一致。

### 业务目标
- **量化指标**：单次查询响应时间 < 5s；支持 102 列表格渲染；列可见性自定义配置持久化
- **目标期限**：2026-Q2 上线

### 涉及系统/模块
- **模块名称**：业财全流程宽表（FullFlowTable）
- **变更类型**：新增
- **对接接口**：`GET /api/reports/full-flow`

## 用户故事

### 故事1
- **角色**：项目经理 / 财务分析人员
- **功能**：在一个 102 列的宽表中同时查看项目基本信息、收入（含税/不含税/分类明细）、收款、支出（总支出/当年/当月，含服务/设备/代收代付子分类）、投资（ICT）、付款等全流程数据
- **收益**：一个界面覆盖项目全生命周期财务数据，无需切换多个报表，数据一致性有保障
- **验收条件**：表格加载后显示 102 列数据，一级表头分组显示（项目基本信息 25 列/收入 25 列/收款 4 列/基本信息 3 列/支出 34 列/投资 7 列/付款 4 列），每行显示一个概算/预算/结算/决算阶段的数据

### 故事2
- **角色**：财务人员
- **功能**：点击"自定义表头"按钮打开 ColumnVisibilityModal，按一级/二级分组选择要显示的列，配置保存后表格重新渲染
- **收益**：用户可以隐藏不需要的列，专注查看关注的数据，减少横向滚动操作
- **验收条件**：点击"自定义表头"后弹出弹窗，显示所有列的可见性开关，勾选/取消后表格实时响应

### 故事3
- **角色**：项目经理
- **功能**：点击主表行右侧"查看"按钮，左侧滑出详情面板（宽度 40%），按分组展示该行所有字段的键值对
- **收益**：大表格行数据以结构化方式展示，避免在小单元格中查看长文本
- **验收条件**：点击"查看"按钮后左侧详情面板展开，显示 5 个分组（项目基本信息/收入/收款/支出/付款），每个分组下按网格布局显示字段

### 故事4
- **角色**：财务人员
- **功能**：每个项目在表格中展示 4 行数据，分别对应概算/预算/结算/决算 4 个阶段，各阶段数据依次排列，颜色区分
- **收益**：一个项目中不同阶段的财务数据对比一目了然，便于发现阶段间的变化
- **验收条件**：概算行蓝色/预算行黄色/结算行绿色/决算行紫色，同一项目的 4 行基本列合并显示（rowSpan=4）

### 故事5
- **角色**：财务人员
- **功能**：点击工具栏"隐藏查询"/"显示查询"按钮，隐藏或显示查询条件区域，增大表格可视区域
- **收益**：查询条件设置完成后可以收起，表格展示更多行数，减少滚动操作
- **验收条件**：点击"隐藏查询"后查询条件卡片消失，表格区域高度增加，底部固定按钮保持显示

## 数据流图

### 接口1：业财全流程数据查询
- **请求路径**：`GET /api/reports/full-flow`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `city` - 类型：字符串；必填：否；来源：查询条件字段 city；校验：枚举值
  - `district` - 类型：字符串；必填：否；来源：查询条件字段 district；校验：枚举值
  - `accountBook` - 类型：字符串；必填：否；来源：查询条件字段 accountBook；校验：枚举值
  - `oppCode` - 类型：字符串；必填：否；来源：查询条件字段 oppCode；校验：非空
  - `projectCode` - 类型：字符串；必填：否；来源：查询条件字段 projectCode；校验：非空
  - `projectType` - 类型：字符串；必填：否；来源：查询条件字段 projectType；校验：枚举值
- **响应字段**：
  - `projects` - 类型：数组；描述：项目列表，每个项目包含 basic/est/plan/settle/final 5 个阶段的数据对象
  - `currentPeriod` - 类型：字符串；描述：当前账期
  - `city` - 类型：字符串；描述：地市
  - `incomeTotalTax` - 类型：数字；描述：项目总金额（含税）
- **存储位置**：数据库表 `full_flow_report`
- **错误码**：
  - `401` - `未授权，请重新登录`
  - `500` - `服务器异常，请稍后重试`

### 数据刷新点
- **刷新时机**：页面加载 / 查询按钮点击后 / 重置后 / 列配置变更后
- **影响字段**：表格所有数据列、详情面板（如已打开）

## 验收标准

### 正常流程
- [ ] **操作**：页面加载 → **预期**：表格显示当前账期的业财全流程数据，共 102 列，三级表头（一级 7 组 + 收入/支出二级子组 + 实际列名）
- [ ] **操作**：点击"展开更多条件" → **预期**：显示"合同信息"查询条件组（7个字段）
- [ ] **操作**：点击"自定义表头" → **预期**：ColumnVisibilityModal 弹窗打开，显示列分组树和可见性开关
- [ ] **操作**：取消勾选"收入"整个分组 → **预期**：表格不再渲染收入相关列，列数减少
- [ ] **操作**：点击第一行"查看"按钮 → **预期**：左侧详情面板展开（40% 宽度），显示 5 个分组的字段键值对
- [ ] **操作**：点击"隐藏查询" → **预期**：查询条件卡片消失，表格区域高度增加
- [ ] **操作**：点击"导出" → **预期**：浏览器下载当前可见列的 Excel 文件

### 异常流程
- [ ] **操作**：查询接口返回 401 → **预期**：页面顶部显示"未授权，请重新登录"
- [ ] **操作**：查询接口返回 500 → **预期**：表格区域显示"服务器异常，请稍后重试"
- [ ] **操作**：网络断开时点击查询 → **预期**：显示"网络异常"提示
- [ ] **操作**：输入的日期范围起 > 止 → **预期**：字段下方显示"开始日期不能大于结束日期"
