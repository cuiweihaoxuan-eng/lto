# 自交付结算能力描述（PRD v10 配套数据）

## Context

自交付结算管理-申请自交付结算弹窗中"基本情况描述"模块的描述类型下拉框，选择二级选项后自动填充到"详细描述"多行文本框。共8个一级分类、42个二级能力，每个二级能力对应一段标准描述文案。

## 数据结构

```ts
interface DescriptionOption {
  value: string;        // 一级分类值
  label: string;        // 一级分类显示名
  children: {
    value: string;        // 二级选项值
    label: string;        // 二级选项显示名
    description: string;  // 详细描述（自动填充到textarea）
  }[];
}
```

## 一级分类与二级能力清单

### 一、软件集成服务

#### 1. 视联网收编
- **AI 能力叠加**：给客户视联网系统添加 AI 功能，提升智能化，高效管理视频监控、减少人工成本。
- **NVR/CVR 调试**：调试监控录像存储设备，确保录像正常录制、回放、导出，保障录像可用可追溯。

#### 2. 服务器
- **私有云电脑部署**：部署专属云桌面，支持员工远程办公，保障数据安全，提升办公灵活性。
- **网管平台部署**：安装调试网管平台，方便客户实时查看网络状态，快速排查网络故障。
- **超融合部署**：整合服务器、存储、网络资源，简化机房布局，提升资源利用率，降低维护成本。
- **底层系统安装**：给服务器安装调试操作系统，确保服务器稳定运行。
- **AI 私有化部署**：将 AI 功能部署在客户本地设备，不依赖外部平台，满足 AI 需求且保障数据隐私。

#### 3. 网络传输（路由/交换/无线/FTTR-B）
- **组网方案撰写**：根据客户需求，撰写专属网络组网方案，明确设备和线路布局，提供实施依据。
- **优化方案撰写**：针对客户网络痛点，撰写优化方案，提出改进措施，提升网络性能。
- **设备调试**：调试路由器、交换机等设备，确保设备正常工作，保障网络稳定连通。
- **设备版本升级**：升级网络设备系统版本，提升设备性能和安全性，避免版本老旧引起的故障风险。
- **业务割接**：网络升级、设备更换时，平稳切换原有业务至新设备/网络。
- **网络排障**：快速排查网络故障（断网、卡顿等），及时修复，减少对客户办公的影响。
- **网络维保**：定期维护网络系统，排查潜在隐患，确保网络长期稳定运行。

#### 4. 网络安全
- **边界防护与访问控制**（防火墙-安全大脑）：调试防火墙，控制内部上网权限，保障网络和数据安全。
- **上网行为管理**：规范员工上网行为，限制不良网站访问、控制下载速度，避免网络卡顿和安全风险。
- **定期扫描与风险评估**（漏洞扫描）：扫描网络、设备漏洞，评估风险并给出修复建议，防范安全事故。
- **互联网 VPN 组网**（非电信VPN）：部署加密 VPN，支持员工远程安全访问公司内网，实现远程办公。
- **认证服务器接入**：接入认证服务器，设置会议访问权限，保障会议内容不泄露、不被干扰。
- **国际网络接入**（需审批）：协助客户申请部署国际网络，保障国际视频会议、跨国业务顺畅。

#### 5. 会议保障
- **重保方案撰写**：为重要会议撰写网络保障方案，明确应急措施，确保会议网络稳定。
- **会场无线调优**：优化会议现场 WiFi，避开干扰，避免卡顿断连，保障参会人员正常使用。
- **5G 信号保障车租赁**：会议现场信号不佳时，租赁 5G 保障车，提供稳定高速网络。
- **应急电源车租赁**：为重要会议提供应急供电，避免停电导致会议中断。

### 二、硬件集成服务

#### 6. 综合布线
- **光缆布放熔接**：布放并熔接光缆，保障大带宽、远距离数据传输，适用于园区、办公楼内部互联。
- **固话线路布放**：布放调试固话线路，确保固话正常通讯，满足办公联系需求。
- **有线点位布放**：安装有线网络接口、布放网线，保障电脑、打印机等设备稳定联网。
- **AP 点位勘测与无线覆盖**：勘测 AP 安装位置，调试无线设备，实现办公区、园区稳定 WiFi 覆盖。
- **门禁系统部署**：安装调试门禁设备，设置访问权限，保障客户区域安全。
- **监控系统部署**：安装调试监控设备，实现实时监控、录像回放，起到安全防范和追溯作用。
- **道闸安装**：安装道闸，调试道闸服务器，调通道闸网络，实现车辆、行人自动识别放行，方便管理。
- **智能家居安装**：安装调试智能设备，实现便捷控制，提升办公舒适度。

#### 7. 机房搬迁
- **服务器搬迁**：安全拆卸、搬运、安装调试服务器，确保数据不丢失、业务不中断。
- **网络设备搬迁**：搬迁调试路由器、交换机等设备，保障网络正常连通。
- **安全设备搬迁**：搬迁调试防火墙等设备，确保网络安全防护不中断。
- **UPS 搬迁**：搬迁 UPS 电源，确保停电时保护设备，避免业务中断。
- **线路整理**：整理机房杂乱线路，规范捆绑、做好标识，方便后续维护排查。
- **拓扑绘制**：摸排网络设备部署情况，绘制机房网络拓扑示意图，排查设备所有接口及配置。
- **线路信息表**：整理线路详细信息做成表格，方便查询和维护。

#### 8. 硬件巡检
- **设备告警状态巡检**：检查硬件设备状态，排查有无异常告警，避免设备故障中断业务。
- **设备维保周期核查**：核对主流品牌设备维保期限，提醒客户续期，避免过保损失。
- **评估报告输出**：巡检后出具报告，明确设备状态、隐患及改进建议，方便客户规划维护。

## UI交互机制

### 描述类型下拉框
- 组件：Radix UI Select + SelectGroup/ SelectLabel
- 宽度：w-72（约 18rem）
- 展示：单下拉框，一级+二级合并（用 SelectGroup 分组）
- 交互：选择二级选项后，自动填充对应 description 到"详细描述"多行文本框

### 详细描述多行文本框
- 默认值：根据选择的二级选项自动填充
- 可编辑：用户可手动修改
- 切换一级分类时：textarea 内容清空，二级下拉重置
