# 政企口径ICT毛利额报表（区县维度）PRD

## 需求背景

### 痛点
- **问题现象**：财务分析人员需要按区县维度查看 ICT 毛利额的同比变化（本年累计 vs 上年同期），当前缺乏统一的区县维度毛利额报表，无法快速对比区域间 ICT 业务盈利能力。
- **发生频率**：高
- **当前 workaround**：从财务系统中按区县导出数据，在 Excel 中手工计算毛利额同比，区域多时工作量大。

### 业务目标
- **量化指标**：报表加载时间 < 3s；本年/上年同期数据同比计算自动完成，毛利额增长率精确显示
- **目标期限**：2026-Q2 上线

### 涉及系统/模块
- **模块名称**：政企口径ICT毛利额报表（区县维度）（IctGrossProfitReport）
- **变更类型**：新增
- **对接接口**：`GET /api/reports/ict-gross-profit`

---

## 用户故事

### 故事1
- **角色**：财务分析人员 / 区域负责人
- **功能**：按地市、区县、账期筛选，查看本年累计和上年同期的 ICT 毛利额数据对比，包括 ICT 收入、物联网收入、产数平台支撑费考核口径、ICT 业务支出考核口径、物联网成本考核口径等指标
- **收益**：一个界面同时展示本年/上年同期 6 个维度 × 2 个时段的数据，无需切换多个报表
- **验收条件**：筛选"杭州市 + 2026-03"，表格展示杭州市及下属区县的毛利额数据，本年列蓝色底/上年列紫色底

### 故事2
- **角色**：区域负责人
- **功能**：对表格中的数值列（ICT 收入/毛利额等）点击列头升序/降序排序，快速识别毛利额最高/最低的区县
- **收益**：快速定位高价值区域和低价值区域，支持区域资源配置优化决策
- **验收条件**：点击"ICT毛利额"列头，表格按毛利额降序排列，排序图标变为降序箭头

### 故事3
- **角色**：财务分析人员
- **功能**：对比查看本年与上年同期的数据差异（通过本年-上年计算），识别业务增长或下滑的区县
- **收益**：在一个报表中直观看到同比变化趋势，无需在两个报表间切换
- **验收条件**：选择账期"2026-03"后，本年/上年列同时显示，对应行的毛利额差异一目了然

## 数据流图

### 接口1：ICT毛利额数据查询
- **请求路径**：`GET /api/reports/ict-gross-profit`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `city` - 类型：字符串；必填：否；来源：查询条件字段 city；校验：枚举值
  - `district` - 类型：字符串；必填：否；来源：查询条件字段 district；校验：枚举值
  - `period` - 类型：字符串；必填：否；来源：查询条件字段 period；校验：YYYY-MM 格式
- **响应字段**：
  - `city` - 类型：字符串；描述：地市
  - `district` - 类型：字符串；描述：区县
  - `period` - 类型：字符串；描述：账期
  - `ictIncomeCur` - 类型：数字；描述：ICT收入（本年累计）
  - `ictCostCur` - 类型：数字；描述：ICT业务支出考核口径（本年累计）
  - `grossProfitCur` - 类型：数字；描述：ICT毛利额（本年累计）
  - `grossProfitLast` - 类型：数字；描述：ICT毛利额（上一年同期）
- **存储位置**：数据库表 `ict_gross_profit_report`
- **错误码**：
  - `401` - `未授权，请重新登录`
  - `500` - `服务器异常，请稍后重试`

### 数据刷新点
- **刷新时机**：页面加载 / 查询按钮点击后 / 重置后
- **影响字段**：表格所有数据列

## 验收标准

### 正常流程
- [ ] **操作**：页面加载 → **预期**：表格显示当前账期的 ICT 毛利额数据，共 15 列，二级表头分组显示（基础维度3列/本年累计6列/上年同期6列）
- [ ] **操作**：选择"杭州市 + 2026-03"后点击查询 → **预期**：表格数据过滤为杭州市及下属区县的毛利额数据
- [ ] **操作**：点击"ICT毛利额"列头 → **预期**：表格按毛利额降序排列，排序图标变为降序箭头
- [ ] **操作**：再次点击"ICT毛利额"列头 → **预期**：切换为升序排列，图标变为升序箭头
- [ ] **操作**：点击表格中的地市名称 → **预期**：展开该地市详情面板（showDetail=true 生效）

### 异常流程
- [ ] **操作**：查询接口返回 401 → **预期**：页面顶部显示"未授权，请重新登录"
- [ ] **操作**：查询接口返回 500 → **预期**：表格区域显示"服务器异常，请稍后重试"
- [ ] **操作**：网络断开时点击查询 → **预期**：显示"网络异常"提示
- [ ] **操作**：选择不存在的账期 → **预期**：表格显示空数据状态提示
