# 关联列账信息 PRD

## 需求背景

### 痛点
- **问题现象**：后向付款确认后需要将付款记录与列账信息关联，当前无界面化操作方式
- **发生频率**：中（每周数次）
- **当前 workaround**：财务人员通过财务系统查询列账凭证号，手动记录关联关系

### 业务目标
- **量化指标**：关联操作时间从 10 分钟/次降至 3 分钟/次
- **目标期限**：2026-Q2

### 涉及系统/模块
- **模块名称**：关联列账信息
- **变更类型**：新增
- **对接接口**：列账信息查询接口（待对接）

---

## 用户故事

### 故事1
- **角色**：财务人员/项目经办人
- **功能**：通过弹窗筛选并勾选列账信息记录，建立与后向付款确认单的关联关系
- **收益**：减少手动录入的工作量，关联关系可追溯
- **验收条件**：输入查询条件后点击搜索，列表正确过滤；勾选记录后点击确认，关联关系被建立

## 数据流图

### 接口1：查询列账信息
- **请求路径**：`GET /api/accounting/items`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `contractCode` - 类型：字符串；必填：否；来源：合同编码；校验：
  - `subjectCode` - 类型：字符串；必填：否；来源：科目编码；校验：
  - `costElementName` - 类型：字符串；必填：否；来源：成本要素名称；校验：
  - `materialNo` - 类型：字符串；必填：否；来源：物料号；校验：
  - `reimbursementNo` - 类型：字符串；必填：否；来源：报账单号；校验：
  - `purchaseVoucherNo` - 类型：字符串；必填：否；来源：采购凭证号；校验：
- **响应字段**：
  - `records[]` - 类型：数组；描述：列账信息列表
  - `records[].id` - 类型：字符串；描述：记录ID
  - `records[].backContractName` - 类型：字符串；描述：后向合同名称
  - `records[].debitAmount` - 类型：数字；描述：借方金额
  - `records[].associatedPaymentAmount` - 类型：数字；描述：已关联付款金额
  - `records[].voucherNo` - 类型：字符串；描述：凭证号
- **存储位置**：财务系统（数据来源）/ LTO本地缓存
- **错误码**：
  - `401` - `无权限`
  - `500` - `列账接口调用失败`

### 接口2：确认关联
- **请求路径**：`POST /api/payment/associate-accounting`
- **请求方法**：POST
- **请求头**：Authorization / Content-Type: application/json
- **请求参数**：
  - `paymentRecordId` - 类型：字符串；必填：是；来源：当前付款确认单ID；校验：
  - `accountingRecordIds[]` - 类型：字符串数组；必填：是；来源：用户勾选的列账记录ID列表；校验：至少选一条
- **响应字段**：
  - `success` - 类型：布尔；描述：是否成功
  - `associatedCount` - 类型：数字；描述：关联的记录数量
- **存储位置**：数据库表 payment_accounting_association
- **错误码**：
  - `400` - `未选择任何记录`
  - `401` - `无权限`
  - `500` - `服务器异常`

### 数据刷新点
- **刷新时机**：点击搜索按钮后 / 点击确认后
- **影响字段**：列表数据 / 后向付款确认单列表

## 验收标准

### 正常流程
- [ ] **操作**：点击某付款确认单的"关联"按钮 → **预期**：弹窗打开，默认尺寸1650×850，查询条件为空，列表显示全部数据（或空状态）
- [ ] **操作**：填写查询条件后点击搜索 → **预期**：列表过滤为符合条件的记录
- [ ] **操作**：点击全选复选框 → **预期**：所有行被勾选
- [ ] **操作**：勾选部分记录后点击确认 → **预期**：弹窗关闭，列表页刷新，该付款确认单已关联选中的列账记录

### 异常流程
- [ ] **操作**：不勾选任何记录直接点击确认 → **预期**：提示"请至少选择一条记录"
- [ ] **操作**：点击重置 → **预期**：所有查询条件清空，列表恢复初始状态
- [ ] **操作**：拖动弹窗标题栏移动位置 → **预期**：弹窗跟随鼠标移动
- [ ] **操作**：拖动弹窗边缘调整大小 → **预期**：弹窗尺寸变化（不小于最小尺寸）
