# 自交付结算统计

## 需求背景
按经营单元、区县、支局维度统计自交付结算情况，支撑业务决策分析。

## 路由信息
- 路径：self-delivery-settlement-stats
- 页面标题：自交付结算统计
- 访问权限：登录

## 功能描述

### 查询条件
| 字段名 | 类型 | 必填 | 默认值 | 展示形式 |
|--------|------|------|--------|----------|
| 账期 | 月份 | 是 | 2026-04 | 月份选择器 |
| 经营单元 | 文本 | 否 | - | 输入框 |
| 区县 | 文本 | 否 | - | 输入框 |

### 下钻机制
- 点击区县行 → 页面刷新到该区县支局明细页
- URL参数：`?district=西湖区&period=2026-04&businessUnit=杭州分公司`
- 支局视图显示"返回上一级"按钮，清除URL参数回到区县视图
- 支局视图"区县"列名改为"支局"

### 列表表头（二级表头）
**一级表头**：项目型 / 小微标品 / 三联单（各占9列）

**二级表头**（每组9列）：
- 数量（1列）
- 可申请（数量/金额 2列）
- 已申请（数量/金额 2列）
- 审核通过（数量/金额 2列）
- 实际发放（数量/金额 2列）

**公共列**：序号、账期、经营单元、区县/支局（4列）

**总列数**：4 + 3×9 = 31列

### 列表数据
- 区县视图：每行展示该区县的3类业务统计
- 支局视图：每行展示该支局的3类业务统计
