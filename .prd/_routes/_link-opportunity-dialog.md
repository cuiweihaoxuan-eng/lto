# 关联已有商机 PRD

## 需求背景

### 痛点
- **问题现象**：商情需要与已有商机建立关联，当前无界面化操作方式；关联操作涉及集团规则（中标前30天不可修改客户名称等）
- **发生频率**：高（每日数次）
- **当前 workaround**：通过后台手动修改关联关系

### 业务目标
- **量化指标**：商情关联商机操作时间从 10 分钟/次降至 2 分钟/次；通过重要提醒减少集团违规判为虚假商机的风险
- **目标期限**：2026-Q2

### 涉及系统/模块
- **模块名称**：关联已有商机
- **变更类型**：新增
- **对接接口**：商机列表查询接口

---

## 用户故事

### 故事1
- **角色**：客户经理/销售人员
- **功能**：通过弹窗搜索并选择已有商机，系统显示重要提醒（中标前30天不可修改客户名称等规则），确认后建立商情与商机的关联
- **收益**：减少手动录入的工作量；重要提醒避免集团违规判定
- **验收条件**：选择商机后点击确认，商情关联上该商机；取消则无变更

## 数据流图

### 接口1：查询商机列表
- **请求路径**：`GET /api/opportunities`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `searchText` - 类型：字符串；必填：否；来源：商机名称输入；校验：
  - `customerName` - 类型：字符串；必填：否；来源：客户名称输入；校验：
  - `linkStatus` - 类型：字符串；必填：否；来源：关联状态下拉；校验：
  - `manager` - 类型：字符串；必填：否；来源：客户经理输入；校验：
- **响应字段**：
  - `opportunities[]` - 类型：数组；描述：商机列表
  - `opportunities[].id` - 类型：字符串；描述：商机ID
  - `opportunities[].name` - 类型：字符串；描述：商机名称
  - `opportunities[].code` - 类型：字符串；描述：商机编码
  - `opportunities[].customerName` - 类型：字符串；描述：客户名称
  - `opportunities[].amount` - 类型：数字；描述：金额（万元）
  - `opportunities[].accountManager` - 类型：字符串；描述：客户经理
  - `opportunities[].hasLinkedInfo` - 类型：布尔；描述：是否已关联商情
- **存储位置**：数据库表 opportunities
- **错误码**：
  - `401` - `无权限`
  - `500` - `服务器异常`

### 接口2：关联商情与商机
- **请求路径**：`POST /api/business-info/link-opportunity`
- **请求方法**：POST
- **请求头**：Authorization / Content-Type: application/json
- **请求参数**：
  - `businessInfoId` - 类型：字符串；必填：是；来源：当前商情ID；校验：
  - `opportunityId` - 类型：字符串；必填：是；来源：用户选择的商机ID；校验：
- **响应字段**：
  - `success` - 类型：布尔；描述：是否成功
- **存储位置**：数据库表 business_info.opportunity_id
- **错误码**：
  - `400` - `参数错误`
  - `401` - `无权限`
  - `409` - `该商机已被其他商情关联`
  - `500` - `服务器异常`

### 数据刷新点
- **刷新时机**：点击查询后 / 确认关联后
- **影响字段**：商机列表数据

## 验收标准

### 正常流程
- [ ] **操作**：点击"关联商机"按钮 → **预期**：弹窗打开，显示重要提醒Banner，商机列表默认显示全部
- [ ] **操作**：输入搜索条件后点击查询 → **预期**：列表过滤为符合条件的商机
- [ ] **操作**：点击某行商机 → **预期**：行高亮，选中该商机，确定按钮变为可用
- [ ] **操作**：再次点击其他行 → **预期**：原行取消选中，新行被选中
- [ ] **操作**：点击确定 → **预期**：打开确认关联弹窗，显示重要提醒
- [ ] **操作**：勾选"我已知悉" → **预期**：确认关联按钮变为可用
- [ ] **操作**：点击确认关联 → **预期**：关联关系建立，弹窗关闭，返回商情详情页

### 异常流程
- [ ] **操作**：不选择商机直接点击确定 → **预期**：按钮禁用，无响应
- [ ] **操作**：点击取消 → **预期**：弹窗关闭，无数据变更
- [ ] **操作**：点击关闭Banner → **预期**：Banner收起，不再显示
- [ ] **操作**：重新打开弹窗 → **预期**：Banner再次显示
- [ ] **操作**：点击重置 → **预期**：所有搜索条件清空，列表恢复初始状态
