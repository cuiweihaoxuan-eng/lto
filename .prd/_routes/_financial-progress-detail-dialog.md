# 财务进度详情弹窗 PRD

## 需求背景

### 痛点
- **问题现象**：项目经理在查看财务进度时间轴时，需要进一步查看某项（收入/支出/收款/付款）的计划或实际明细，当前系统无法直接在时间轴节点上展开，需要跳转其他模块或导出 Excel 再分析。
- **发生频率**：中
- **当前 workaround**：用户通过财务系统单独查询每类明细，或导出 Excel 后手工整理。

### 业务目标
- **量化指标**：弹窗加载时间 < 2s；支持 30+ 列表格渲染
- **目标期限**：2026-Q2 上线

### 涉及系统/模块
- **模块名称**：财务进度详情弹窗（FinancialProgressDetailDialog）
- **变更类型**：新增
- **对接接口**：`GET /api/reports/financial-progress/detail`

## 用户故事

### 故事1
- **角色**：项目经理 / 财务人员
- **功能**：在财务进度时间轴上点击某个"收入计划"/"收入"/"支出计划"/"支出"/"收款计划"/"收款"/"付款计划"/"付款"节点，弹出详情弹窗展示该类别的明细数据
- **收益**：一个弹窗内展示该项目该类别的所有明细行，无需跳转其他页面，数据量大时支持横向滚动
- **验收条件**：点击"收入计划"节点后弹窗显示收入计划明细表格，共25列，点击"支出实际"节点后显示支出实际明细，共26列

### 故事2
- **角色**：项目经理
- **功能**：弹窗可自由拖拽移动位置，方便在多显示器场景下参考对比数据
- **收益**：弹窗不遮挡主时间轴内容，可将弹窗拖到屏幕一侧同时查看时间轴
- **验收条件**：按住弹窗头部拖拽，弹窗跟随鼠标移动，松开后保持在目标位置

### 故事3
- **角色**：财务分析人员
- **功能**：弹窗中展示该项目该类别的基本信息（类型/细项名称/金额/时间/进度比例），便于在查看明细前快速了解概况
- **收益**：概览信息帮助用户快速判断是否找到了目标明细，减少无效滚动
- **验收条件**：弹窗顶部显示5个概览字段，点击"关闭"按钮后弹窗消失

## 数据流图

### 接口1：财务进度详情数据查询
- **请求路径**：`GET /api/reports/financial-progress/detail`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `projectCode` - 类型：字符串；必填：是；来源：event 中携带的项目编码；校验：非空
  - `lineType` - 类型：字符串；必填：是；来源：父组件传入 lineType；校验：枚举值（收入/支出/收款/付款）
  - `eventType` - 类型：字符串；必填：是；来源：event.type；校验：枚举值（plan/actual）
- **响应字段**：
  - `contractCode` - 类型：字符串；描述：合同编码
  - `contractName` - 类型：字符串；描述：合同名称
  - `amount` - 类型：数字；描述：金额
  - `profitCenter` - 类型：字符串；描述：利润中心
  - `costCenter` - 类型：字符串；描述：成本中心
- **存储位置**：数据库表 `financial_progress_detail`
- **错误码**：
  - `401` - `未授权，请重新登录`
  - `404` - `未找到该项目详情数据`
  - `500` - `服务器异常，请稍后重试`

### 数据刷新点
- **刷新时机**：弹窗打开时加载（单次加载，不轮询）
- **影响字段**：弹窗内表格数据

## 验收标准

### 正常流程
- [ ] **操作**：点击时间轴"收入计划"节点 → **预期**：弹窗打开，显示"收入计划 - 详细信息"标题，顶部概览区显示类型=计划、细项名称、金额、时间、进度比例，明细表格显示25列收入计划数据
- [ ] **操作**：点击时间轴"支出实际"节点 → **预期**：弹窗显示"支出实际 - 详细信息"，概览区类型=实际，明细表格显示26列支出数据
- [ ] **操作**：按住弹窗头部拖拽到屏幕右侧 → **预期**：弹窗跟随鼠标移动，松开后保持在右侧
- [ ] **操作**：点击弹窗外部遮罩 → **预期**：弹窗关闭，数据释放
- [ ] **操作**：点击弹窗右上角关闭图标 → **预期**：弹窗关闭
- [ ] **操作**：点击弹窗底部"关闭"按钮 → **预期**：弹窗关闭

### 异常流程
- [ ] **操作**：接口返回 404 → **预期**：弹窗显示"未找到该项目详情数据"提示
- [ ] **操作**：接口返回 500 → **预期**：弹窗表格区域显示"服务器异常，请稍后重试"
- [ ] **操作**：网络断开时打开弹窗 → **预期**：显示"网络异常"提示，弹窗显示加载失败状态
