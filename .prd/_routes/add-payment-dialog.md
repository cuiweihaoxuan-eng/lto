# 添加SAP付款明细 PRD

## 需求背景

### 痛点
- **问题现象**：后向付款确认需要从SAP系统选择付款明细记录，当前无界面化操作方式
- **发生频率**：中（每周数次）
- **当前 workaround**：财务人员通过SAP系统手动复制凭证号，再粘贴到LTO系统

### 业务目标
- **量化指标**：后向付款关联操作时间从 15 分钟/次降至 5 分钟/次
- **目标期限**：2026-Q2

### 涉及系统/模块
- **模块名称**：添加SAP付款明细
- **变更类型**：新增
- **对接接口**：SAP付款明细查询接口（待对接）

---

## 用户故事

### 故事1
- **角色**：财务人员/项目经办人
- **功能**：通过弹窗筛选并勾选SAP系统中的付款明细记录，批量添加到后向付款确认单
- **收益**：减少手动录入凭证号的工作量，避免录入错误
- **验收条件**：输入查询条件后点击搜索，列表正确过滤；勾选记录后点击确认，选中记录被添加到付款确认单

## 数据流图

### 接口1：查询SAP付款明细
- **请求路径**：`GET /api/sap/payment-records`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `startDate` - 类型：字符串（日期）；必填：否；来源：查询起止时间-开始；校验：
  - `endDate` - 类型：字符串（日期）；必填：否；来源：查询起止时间-结束；校验：
  - `counterpartyName` - 类型：字符串；必填：否；来源：对方名称；校验：
  - `paymentAmount` - 类型：数字；必填：否；来源：付款金额；校验：
  - `sapVoucherNo` - 类型：字符串；必填：否；来源：SAP凭证号；校验：
  - `financeReimbursementNo` - 类型：字符串；必填：否；来源：财辅报账单号；校验：
- **响应字段**：
  - `records[]` - 类型：数组；描述：付款明细记录列表
  - `records[].id` - 类型：字符串；描述：记录ID
  - `records[].projectName` - 类型：字符串；描述：项目名称
  - `records[].sapVoucherNo` - 类型：字符串；描述：SAP凭证号
  - `records[].debitAmount` - 类型：数字；描述：借方金额
  - `records[].creditAmount` - 类型：数字；描述：贷方金额
  - `records[].counterpartyName` - 类型：字符串；描述：对方名称
- **存储位置**：SAP系统（数据来源）/ LTO本地缓存
- **错误码**：
  - `401` - `无权限`
  - `500` - `SAP接口调用失败`

### 接口2：确认添加付款明细
- **请求路径**：`POST /api/payment/confirm-records`
- **请求方法**：POST
- **请求头**：Authorization / Content-Type: application/json
- **请求参数**：
  - `recordIds[]` - 类型：字符串数组；必填：是；来源：用户勾选的记录ID列表；校验：至少选一条
- **响应字段**：
  - `success` - 类型：布尔；描述：是否成功
  - `addedCount` - 类型：数字；描述：添加的记录数量
- **存储位置**：数据库表 payment_confirmation_records
- **错误码**：
  - `400` - `未选择任何记录`
  - `401` - `无权限`
  - `500` - `服务器异常`

### 数据刷新点
- **刷新时机**：点击搜索按钮后 / 点击确认后
- **影响字段**：列表数据 / 付款确认单列表

## 验收标准

### 正常流程
- [ ] **操作**：点击"新增付款确认" → **预期**：弹窗打开，默认尺寸1600×850，查询条件为空，列表显示全部数据（或空状态）
- [ ] **操作**：填写查询条件后点击搜索 → **预期**：列表过滤为符合条件的记录
- [ ] **操作**：点击全选复选框 → **预期**：所有行被勾选
- [ ] **操作**：勾选部分记录后点击确认 → **预期**：弹窗关闭，列表页刷新，添加了选中的记录

### 异常流程
- [ ] **操作**：不勾选任何记录直接点击确认 → **预期**：提示"请至少选择一条记录"
- [ ] **操作**：结束日期早于开始日期 → **预期**：前端校验提示日期范围不正确
- [ ] **操作**：点击重置 → **预期**：所有查询条件清空，列表恢复初始状态
- [ ] **操作**：拖动弹窗标题栏移动位置 → **预期**：弹窗跟随鼠标移动
- [ ] **操作**：拖动弹窗边缘调整大小 → **预期**：弹窗尺寸变化（不小于最小尺寸）
