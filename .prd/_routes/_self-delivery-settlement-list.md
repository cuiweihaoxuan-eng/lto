# 自交付结算清单

## 需求背景
提供自交付结算单明细查询，支撑业务人员快速定位需要查看或跟踪的结算单。

## 路由信息
- 路径：self-delivery-settlement-list
- 页面标题：自交付结算清单
- 访问权限：登录

## 功能描述

### 查询条件
| 字段名 | 类型 | 必填 | 默认值 | 展示形式 |
|--------|------|------|--------|----------|
| 经营单元 | 文本 | 否 | - | 输入框 |
| 支局 | 文本 | 否 | - | 输入框 |
| 业务类型 | 枚举 | 否 | 全部 | 下拉选择（项目型/小微标品/三联单） |
| 结算状态 | 枚举 | 否 | 全部 | 下拉选择（审核中/审核通过/审核驳回/发放中/可发放/已发放） |
| 合同/小微工单/三联单编码 | 文本 | 否 | - | 输入框 |
| 申请人 | 文本 | 否 | - | 输入框 |
| 申请时间起 | 日期 | 否 | - | 日期选择器 |
| 申请时间止 | 日期 | 否 | - | 日期选择器 |

### 列表字段
| 字段名 | 类型 | 展示形式 |
|--------|------|----------|
| 序号 | 数字 | 居中 |
| 结算单名称 | 文本 | 文本 |
| 结算单编码 | 文本 | 文本 |
| 经营单元 | 文本 | 文本 |
| 支局 | 文本 | 文本 |
| 业务类型 | 枚举 | 标签（项目型/小微标品/三联单） |
| 合同/小微工单/三联单编码 | 文本 | 文本 |
| 前向自交付金额 | 金额 | 右对齐 |
| 已申请自交付金额 | 金额 | 右对齐蓝色 |
| 本次申请自交付金额 | 金额 | 右对齐绿色加粗 |
| 发放人员 | 数组 | 人员标签组 |
| 申请日期 | 日期 | 居中 |
| 申请人 | 文本 | 文本 |
| 结算状态 | 枚举 | 状态徽章 |

### 操作按钮
- 重置、查询
- 导出（导出当前列表为Excel）
