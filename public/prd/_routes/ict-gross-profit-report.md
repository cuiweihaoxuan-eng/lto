# 政企口径ICT毛利额报表 PRD

## 需求背景
展示政企口径ICT毛利额统计数据，按区县维度分析ICT业务收支和毛利情况。

## 前端页面描述
- 组件：IctGrossProfitReport
- 位置：作为页面内容显示

## 功能描述

### 页面布局
| 区域 | 组件 | 说明 |
|------|------|------|
| 查询区 | ReportTemplate | 按维度筛选 |
| 操作区 | 按钮组 | 导出、刷新 |
| 数据表格 | ReportTemplate | 16列表格 |

### 查询字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 市级 | Select | 否 | 全部 | - |
| 账期 | Select | 否 | 空 | - |

### 表格列（16列，3个分组）
| 分组 | 列名 |
|------|------|
| 基础维度（3列） | 市级、区县、账期 |
| 本年累计（6列） | ICT收入、物联网收入、产数平台支撑费考核口径、ICT业务支出考核口径、物联网成本考核口径、ICT毛利额 |
| 上年同期（6列） | 同上指标的上年同期数据 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 查询 | 操作区 | Primary | 执行筛选查询 |
| 重置 | 操作区 | Outline | 重置筛选条件 |
| 导出数据 | 操作区 | Outline | 导出报表数据 |
