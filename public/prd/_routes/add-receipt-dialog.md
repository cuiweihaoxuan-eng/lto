# 添加银行流水明细弹窗 PRD

## 需求背景
新增前向收款确认记录，录入银行流水信息。

## 前端页面描述
- 组件：AddReceiptDialog
- 位置：作为弹窗显示
- 宽度：max-w-2xl

## 功能描述

### 表单字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 项目选择 | Select | 是 | 空 | 联动加载合同 |
| 合同选择 | Select | 是 | 空 | 根据项目加载 |
| 银行流水号 | Input | 是 | 空 | - |
| 收款认领金额 | number | 是 | 空 | - |
| 认领日期 | date | 是 | 空 | - |
| 资金用途 | Select | 是 | 销账 | 销账/预存款 |
| 认领人 | Input | 是 | 当前用户 | - |
| 利润中心 | Select | 是 | 空 | - |
| 成本中心 | Select | 是 | 空 | - |

### 联动逻辑
1. **选择项目后**：合同选择项加载该项目下的合同列表
2. **资金用途默认**：默认为"销账"

### 操作按钮
| 按钮名称 | 样式 | 说明 |
|----------|------|------|
| 提交 | Primary | 提交表单并关闭 |
| 取消 | Outline | 关闭弹窗 |
