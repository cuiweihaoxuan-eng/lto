# 人员自交付结算统计

## 需求背景
按人员维度统计各状态（审核中/审核通过/待发放/可发放/已发放）的单数与金额。

## 路由信息
- 路径：person-settlement-stats
- 页面标题：人员自交付结算统计
- 访问权限：登录

## 功能描述

### 查询条件
| 字段名 | 类型 | 必填 | 默认值 | 展示形式 |
|--------|------|------|--------|----------|
| 经营单元 | 文本 | 否 | - | 输入框 |
| 支局 | 文本 | 否 | - | 输入框 |
| 结算人员 | 文本 | 否 | - | 输入框 |

### 列表表头（双行表头）
**左侧公共列**（双行）：序号、经营单元、支局、姓名、电话、工号、部门

**右侧5状态矩阵**（每状态2列：单数+金额）：
- 审核中（蓝底）
- 审核通过（绿底）
- 待发放（灰底）
- 可发放（橙底）
- 已发放（翠绿底）

**总列数**：7（公共）+ 5×2（状态）= 17列

### 列表数据
- 每人一行，展示该人员在5个状态下的单数和金额
- 颜色编码：审核中=蓝、审核通过=绿、待发放=灰、可发放=橙、已发放=翠绿
