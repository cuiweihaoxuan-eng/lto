# 添加SAP付款明细弹窗 PRD

## 需求背景
新增后向付款确认记录，录入SAP付款信息。

## 前端页面描述
- 组件：AddPaymentDialog
- 位置：作为弹窗显示

## 功能描述

### 表单字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 项目选择 | Select | 是 | 空 | 联动加载合同 |
| 合同选择 | Select | 是 | 空 | 根据项目加载 |
| SAP付款单号 | Input | 是 | 空 | - |
| 付款金额 | number | 是 | 空 | - |
| 付款日期 | date | 是 | 空 | - |
| 资金用途 | Select | 是 | 空 | - |
| 付款人 | Input | 是 | 当前用户 | - |

### 操作按钮
| 按钮名称 | 样式 | 说明 |
|----------|------|------|
| 提交 | Primary | 提交表单并关闭 |
| 取消 | Outline | 关闭弹窗 |
