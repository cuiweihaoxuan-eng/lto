# ICT项目预算明细 PRD

## 需求背景

### 痛点
- **问题现象**：ICT 项目经理需要查看各子项目的预算使用情况（总体预算/在途预算/累计支出/可用预算），当前数据分散在多个系统中，无法快速获取可用预算余额。
- **发生频率**：高
- **当前 workaround**：项目经理通过财务系统单独查询各子项目的预算执行情况，手工计算可用预算，效率低且数据不实时。

### 业务目标
- **量化指标**：报表加载时间 < 3s；预算执行进度自动计算，差异率精确到小数点后两位
- **目标期限**：2026-Q2 上线

### 涉及系统/模块
- **模块名称**：ICT项目预算明细（IctBudgetDetail）
- **变更类型**：新增
- **对接接口**：`GET /api/reports/ict-budget-detail`

## 用户故事

### 故事1
- **角色**：ICT 项目经理 / 预算管理员
- **功能**：按地市、区县、账期、项目编码、预算责任中心等多维度筛选，查看各子项目的预算执行情况（不含税金额/税率/含税金额/总体预算/在途预算/累计支出/可用预算）
- **收益**：一个界面掌握所有 ICT 子项目的预算使用进度，便于提前规划采购和支出安排
- **验收条件**：筛选"杭州 + 2026-03"，表格展示杭州地区所有 ICT 子项目，可用预算列为 0 或负值时高亮红色告警

### 故事2
- **角色**：财务分析人员
- **功能**：按直接成本类型（系统集成支出/软件开发支出/运维服务支出）和预算指标维度查看预算执行情况
- **收益**：识别不同成本类型的预算消耗速度，支持预算调整决策
- **验收条件**：切换成本类型筛选条件后，表格数据按成本类型过滤，合计行同步更新

### 故事3
- **角色**：项目经理
- **功能**：在表格中查看子项目的状态（未关闭/实施中/已关闭），快速筛选出仍在执行中的项目
- **收益**：过滤掉已关闭项目，专注管理在执行项目的预算
- **验收条件**：项目状态列显示彩色状态标签（未关闭=蓝色/实施中=橙色/已关闭=灰色）

## 数据流图

### 接口1：ICT项目预算明细数据查询
- **请求路径**：`GET /api/reports/ict-budget-detail`
- **请求方法**：GET
- **请求头**：Authorization
- **请求参数**：
  - `city` - 类型：字符串；必填：否；来源：查询条件字段 city；校验：枚举值
  - `district` - 类型：字符串；必填：否；来源：查询条件字段 district；校验：枚举值
  - `period` - 类型：字符串；必填：否；来源：查询条件字段 period；校验：YYYY-MM 格式
  - `mainProjectCode` - 类型：字符串；必填：否；来源：查询条件字段 mainProjectCode；校验：非空
  - `costType` - 类型：字符串；必填：否；来源：查询条件字段 costType；校验：枚举值
  - `budgetCenter` - 类型：字符串；必填：否；来源：查询条件字段 budgetCenter；校验：枚举值
  - `subProjectStatus` - 类型：字符串；必填：否；来源：查询条件字段 subProjectStatus；校验：枚举值
- **响应字段**：
  - `period` - 类型：字符串；描述：账期
  - `mainProjectCode` - 类型：字符串；描述：主项编码
  - `subProjectCode` - 类型：字符串；描述：子项目编码
  - `costType` - 类型：字符串；描述：直接成本类型
  - `totalBudget` - 类型：数字；描述：子项目总体预算
  - `onTheWayBudget` - 类型：数字；描述：在途总体预算金额
  - `cumulativeExpense` - 类型：数字；描述：累计支出
  - `availableBudget` - 类型：数字；描述：可用总体预算
- **存储位置**：数据库表 `ict_budget_detail`
- **错误码**：
  - `401` - `未授权，请重新登录`
  - `500` - `服务器异常，请稍后重试`

### 数据刷新点
- **刷新时机**：页面加载 / 查询按钮点击后 / 重置后
- **影响字段**：表格所有数据列

## 验收标准

### 正常流程
- [ ] **操作**：页面加载 → **预期**：表格显示当前账期的 ICT 项目预算明细数据，共 17 列，二级表头分组显示（项目信息/金额信息/预算信息/执行情况/状态）
- [ ] **操作**：选择"杭州 + 2026-03 + 系统集成支出"后点击查询 → **预期**：表格数据过滤为杭州地区、2026-03 账期、系统集成支出类型的项目
- [ ] **操作**：展开"项目金额"查询条件组 → **预期**：显示5个金额范围查询字段
- [ ] **操作**：输入可用预算范围"0-10000"后查询 → **预期**：表格仅显示可用预算在0-10000之间的项目
- [ ] **操作**：点击表格中的子项目编号 → **预期**：展开该项目详情面板（showDetail=true 生效）
- [ ] **操作**：可用预算为负的行 → **预期**：可用预算列显示红色文字

### 异常流程
- [ ] **操作**：查询接口返回 401 → **预期**：页面顶部显示"未授权，请重新登录"
- [ ] **操作**：查询接口返回 500 → **预期**：表格区域显示"服务器异常，请稍后重试"
- [ ] **操作**：网络断开时点击查询 → **预期**：显示"网络异常"提示
- [ ] **操作**：输入的数值范围起值 > 止值 → **预期**：字段下方显示"起始值不能大于结束值"
