# ICT项目分成异常报表 PRD

## 需求背景
监控ICT项目分成异常情况，包含月报和年报两种模式，帮助管理者发现分成问题。

## 前端页面描述
- 组件：IctShareAbnormalReport
- 位置：作为页面内容显示

## 功能描述

### 页面布局
| 区域 | 组件 | 说明 |
|------|------|------|
| Tab切换 | 按钮组 | 月报/年报 |
| 查询区 | 表单 | 基础条件+项目信息+更多条件 |
| 操作区 | 按钮组 | 导出、刷新 |
| 数据表格 | 分组表格 | 月报28列/年报33列 |

### Tab结构
| Tab名称 | 列数 | 说明 |
|---------|------|------|
| 月报 | 28列 | 按月统计数据 |
| 年报 | 33列 | 按年统计数据 |

### 查询字段
| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| 时间范围 | DateRangePicker | 否 | 当前月度 | 月报时默认当月 |
| 项目名称 | Input | 否 | 空 | - |
| 省份 | Select | 否 | 全部 | - |
| 分成状态 | Select | 否 | 全部 | 正常/异常/待稽核 |
| 更多条件 | 折叠面板 | 否 | 空 | 展开更多筛选条件 |

### 表格分组（数据分组展示）
| 分组 | 说明 |
|------|------|
| 项目基本信息 | 项目编号、名称、省份等 |
| 前向合同信息 | 前向合同金额、签订时间等 |
| 分成合同信息 | 分成比例、分成金额等 |
| 收入/减收数据 | 收入确认、减收金额等 |
| 进度信息 | 完成进度、确认进度等 |
| 稽核结果 | 异常类型、处理状态等 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 查询 | 操作区 | Primary | 执行筛选查询 |
| 重置 | 操作区 | Outline | 重置筛选条件 |
| 导出数据 | 操作区 | Outline | 导出报表数据 |
| 刷新 | 操作区 | Outline | 刷新列表 |

### 分成状态Badge
| 状态值 | 颜色 | 说明 |
|--------|------|------|
| 正常 | 绿色 | 分成数据正常 |
| 异常 | 红色 | 分成数据异常 |
| 待稽核 | 橙色 | 待稽核确认 |
