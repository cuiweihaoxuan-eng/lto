# 人员自交付结算清单

## 需求背景
按人员维度展示自交付结算明细，支撑人员级的结算查询与跟踪。

## 路由信息
- 路径：person-settlement-list
- 页面标题：人员自交付结算清单
- 访问权限：登录

## 功能描述

### 查询条件
| 字段名 | 类型 | 必填 | 默认值 | 展示形式 |
|--------|------|------|--------|----------|
| 经营单元 | 文本 | 否 | - | 输入框 |
| 支局 | 文本 | 否 | - | 输入框 |
| 结算人员 | 文本 | 否 | - | 输入框 |
| 业务类型 | 枚举 | 否 | 全部 | 下拉选择 |
| 结算状态 | 枚举 | 否 | 全部 | 下拉选择（审核中/审核通过/审核驳回/待发放/可发放/已发放） |
| 合同/小微工单/三联单编码 | 文本 | 否 | - | 输入框 |
| 申请人 | 文本 | 否 | - | 输入框 |
| 申请时间起 | 日期 | 否 | - | 日期选择器 |
| 申请时间止 | 日期 | 否 | - | 日期选择器 |
