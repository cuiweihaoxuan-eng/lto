# 流程列表 PRD

## 需求背景
展示流程配置列表，支持流程的新增、编辑、删除、启用/禁用操作。

## 前端页面描述
- 组件：ProcessList
- 位置：作为页面内容显示

## 功能描述

### 表格/列表
| 列名 | 宽度 | 可排序 | 对齐 | 说明 |
|------|------|--------|------|------|
| 序号 | 60px | 否 | center | - |
| 流程名称 | 160px | 否 | left | - |
| 流程类型 | 100px | 否 | center | Badge |
| 关联项目 | 120px | 否 | left | - |
| 状态 | 80px | 否 | center | Badge |
| 创建时间 | 120px | 否 | center | - |
| 操作 | 120px | 否 | center | 编辑/删除/启用 |

### 操作按钮
| 按钮名称 | 位置 | 样式 | 说明 |
|----------|------|------|------|
| 新增流程 | 操作区 | Primary | 打开新增流程弹窗 |
| 刷新 | 操作区 | Outline | 刷新列表 |
| 编辑 | 表格操作列 | text | 进入编辑视图 |
| 删除 | 表格操作列 | text | 删除流程 |
| 启用/禁用 | 表格操作列 | text | 切换状态 |

### 状态Badge
| 状态值 | 颜色 | 说明 |
|--------|------|------|
| 启用 | 绿色 | 流程已启用 |
| 禁用 | 灰色 | 流程已禁用 |
